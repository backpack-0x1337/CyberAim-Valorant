
aimlab - v1 aimlab-model-200
==============================

This dataset was exported via roboflow.ai on February 5, 2022 at 11:22 AM GMT

It includes 199 images.
Target are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Resize to 416x416 (Fill (with center crop))

No image augmentation techniques were applied.
